Welcome to Recall

AI-powered incident code resolution with local-first intelligence. Integrate your organization to sync real tickets and train Recall on resolved fixes.

What Recall Does

Code-Level Incident Recommendations

Recall learns from resolved incidents and the code changes that fixed them. It retrieves similar cases and recommends ranked, evidence-backed fixes.

Hybrid retrieval Patch evidence Confidence aware Local-first

Under The Hood

Technology Inside Recall

Hybrid retrieval combines lexical match, embeddings, and structured signals so recommendations are specific and explainable.

Frontend: JavaScript workspace with local-first state.
Integrations: Azure DevOps and Jira profile-based intake.
Retrieval: BM25 + semantic + signal overlap.
Embeddings: Ollama local models for semantic recall.
Ranking: Weighted scoring with confidence-aware fallback.
Output: Ranked fixes with similar-incident evidence.

Data Privacy: Recommendations, ranking, and analytics are computed locally on your machine. Optional ticket sync (Azure DevOps/Jira) only runs when you configure and trigger it. Ticket history, training data, and feedback weights are stored locally in your browser's localStorage. Integration tokens are kept in session storage and are not persisted across browser restarts. You may clear all saved data at any time from the History tab.
